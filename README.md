Requirements
------------
- Python >= 3.11.6
- CC --std=c99 -lm (Clang or GCC).

Running
-------
```
$ git clone https://github.com/mahmoudElshimi/vector_space
$ cd vector_space
$ python vec.py
$ cc --std=c99 -lm -Wall -o vec vec.c && ./vec
```
